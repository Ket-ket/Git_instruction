# **Инструкция по работе с Git**
 Создание пустого репозитория (*[документация](https://git-scm.com/docs/git-init "git init")*):
 ```sh
git init
 ```
 Просмотр текущего состояния репозитория (*[документация](https://git-scm.com/docs/git-status)*):
```sh
git status
```
